To create a repository on Quay.io, you must use an authenticated user or robot account.

You are not signed into Quay.io in the Docker CLI

To login, execute docker login quay.io and give it your Quay.io user or robot account credentials. 
docker login quay.io
Login against server at https://quay.io/v1/
Username: myusername
Password: (my password)
Email: myemail@example.com

You do not have a user account

To create a new user, visit the Sign Up page and fill out the form under 'Create Account'.
