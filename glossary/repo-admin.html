--- layout: doc sublayout: glossary title: Repository Admin Panel ---

The repository admin panel provides the ability to manage all aspects of a repository.

Access the repository admin panel

The repository admin panel can be accessed by visiting the repository's page and clicking on the blue gear icon next to the repository's name.

Permissions

The permissions tab lists all the permissions for the repository. Any user, robot account, team or access token with permissio on the repository with be listed here.

Defined Permissions

NameAbilities
Read Can view the repository and pull the repository
Write Can view, pull and push the repository
Admin Has full admin control of the repository

Adding a permission

To add a permission, enter a username, robot account or team name into the text field and hit enter. Alternatively, click the down arrow to see a list of teams/robot accounts.

Changing a permission

To change a permission for a user, robot account or team, click the permission next to that entity.

Deleting a permission

To delete a permission, click the icon and then click Delete

Webhooks

The webhooks tab allows for the creation and management of webhooks.
